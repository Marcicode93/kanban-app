Start and stop scripts for the Docker container.

| Script       | Platform      |
|--------------|---------------|
| `start.sh`   | Mac / Linux   |
| `stop.sh`    | Mac / Linux   |
| `start.ps1`  | Windows       |
| `stop.ps1`   | Windows       |

Container name: `pm-kanban`. Default port: `8000` (override with `PORT` env var).

# Project Management MVP

Local Kanban board with AI assistant. Runs in Docker.

## Start

Mac / Linux:

```bash
./scripts/start.sh
```

Windows (PowerShell):

```powershell
.\scripts\start.ps1
```

Open http://localhost:8000

## Stop

Mac / Linux:

```bash
./scripts/stop.sh
```

Windows:

```powershell
.\scripts\stop.ps1
```

## API

- `GET /api/health` — health check
